# Weather (Flutter Version)

- Simple Android/iOS App using Flutter for displaying week weather.

- Topics:
  * Layered Architecture: 3 Layers : (Presentation, Domain, Data) Layers.
  * Presentation Layer:
    * Flutter Widgets.
    * MVI Presentation Architecture Pattern.
  * Domain Layer:
    * UseCases.
  * Data Layer:
    * Repository.
    * Models.
    * APIs: Dio.
